## Overview

WiFi Kings is a multi-provider telecom acquisition engine built to capture and qualify high-intent internet customers at scale. The system routes visitors through an address-based eligibility check, matches them to available providers, and hands off qualified leads directly into CRM automation — without manual intervention.

## System Architecture

The core funnel is built around address-level availability logic. A visitor enters their address, the system checks eligibility across multiple fiber and wireless providers in real time, and routes them to the correct offer. No dead ends. No manual qualification.

## Stack

Next.js, Lead Routing Logic, CRM Handoff, Funnel Architecture, Analytics

## Flow

Address check → Qualification → Provider routing → Appointment request → CRM automation

## Results

Built to scale across regions and dealer networks without rebuilding the core funnel. The system handles volume increases without degradation in routing accuracy or conversion rate.
