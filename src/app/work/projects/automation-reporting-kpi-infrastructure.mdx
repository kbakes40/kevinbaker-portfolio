## Overview

A backend automation and reporting infrastructure built to eliminate manual data work and surface real-time KPIs across operations. The system integrates multiple data sources, automates reporting exports, and triggers alerts when metrics fall outside defined thresholds.

## System Architecture

Data from multiple sources is synced into a unified backend. KPI tracking logic runs on a scheduled basis and on-demand. Alerts fire automatically when performance metrics deviate. Reporting exports are generated and distributed without manual intervention.

## Stack

Google Sheets, Excel, APIs, Webhooks, Scripts, Dashboards

## Flow

Data sync → KPI tracking → Alerts → Automated reporting exports

## Results

Reduced manual reporting time to near zero and improved operational visibility through real-time dashboards and automated alert logic.
