## Overview

AI Order Automation System

A repeatable AI powered order automation system designed for businesses that need faster, more structured ordering experiences across messaging, websites, and custom digital channels. It handles product navigation, recommendations, order intake, customer flow management, and staff notifications through flexible backend automation and structured logic.

## System Architecture

The bot operates on a structured decision tree. Customers navigate through menu options, receive recommendations based on their selections, and submit orders directly through the chat interface. Webhooks fire to the backend on order submission, triggering staff notifications and fulfillment tracking updates.

## Stack

Telegram Bot API, Node.js, Webhooks, Database Logic

## Flow

Menu → Recommendation → Order intake → Staff notification → Fulfillment tracking

## Results

Eliminated manual order intake and reduced fulfillment errors by routing all orders through a structured, automated flow with backend confirmation logic.
